# 🧩 Sudoku Generator

Tarayıcıda çalışan, sıfırdan yazılmış interaktif Sudoku uygulaması.

🌐 **[Canlı Demo → arcahmet.github.io/Sudoku-Generator](https://arcahmet.github.io/Sudoku-Generator)**

---

## 📋 Hakkında

Sudoku Generator, herhangi bir kurulum veya hesap gerektirmeden tarayıcıda oynanabilen tam özellikli bir Sudoku uygulamasıdır. Bulmacalar algoritmik olarak üretilir ve her oyun benzersizdir.

## 🚀 Özellikler

- ♟️ **3 zorluk seviyesi** — Kolay, Orta, Zor
- ⏱️ **Zamanlayıcı** — Çözme süresini takip eder
- 💡 **İpucu sistemi** — Tıkanan anlarda yardım alabilirsin
- ✅ **Otomatik doğrulama** — Hatalı girişler anında işaretlenir
- 📱 **Responsive tasarım** — Masaüstü ve mobilde sorunsuz çalışır
- 🎨 **Sade arayüz** — Dikkat dağıtmayan, odak odaklı tasarım

## 🛠️ Teknolojiler

Harici kütüphane veya framework kullanılmamıştır.

| | |
|---|---|
| **HTML5** | Yapı |
| **CSS3** | Stil ve responsive layout |
| **Vanilla JavaScript** | Sudoku algoritması ve oyun mantığı |

## 🧮 Nasıl Çalışır?

1. Backtracking algoritmasıyla geçerli bir Sudoku tahtası üretilir
2. Zorluk seviyesine göre belirli sayıda hücre gizlenir
3. Kullanıcının girişleri anlık olarak doğrulanır

## 🖥️ Yerel Çalıştırma

```bash
git clone https://github.com/arcahmet/Sudoku-Generator.git
cd Sudoku-Generator
# index.html dosyasını tarayıcıda aç — başka bir şey gerekmez!
```
